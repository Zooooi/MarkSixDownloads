應用程式簡介
Application Overview

Mark 6 Stats 六合彩統計助手 基於去中心化理念，將所有紀錄整合為本地數據模型，能快速查詢統計，並支援號碼篩選、紀錄管理與自動兌獎功能，是一款方便用戶選號及兌獎的實用工具。
Mark 6 Stats Lottery Analysis Assistant is built on a decentralized concept, integrating all records into a local data model. It enables fast statistical queries and supports number filtering, record management, and automatic prize checking, making it a convenient tool for users to select numbers and verify results.
